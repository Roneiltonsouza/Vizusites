# 🚀 Vizusites - GitHub Repo Engine v5.0

O **Vizusites** é uma ferramenta de automação de hospedagem que permite transformar códigos HTML brutos em sites funcionais instantaneamente, utilizando a API do GitHub.

## 🛠️ Funcionalidades
- **Login Persistente:** O sistema lembra de você.
- **Auto-Repo:** Cria repositórios reais diretamente na sua conta.
- **Deploy Instantâneo:** Utiliza CDN para visualização imediata do HTML.
- **Design Neon:** Interface moderna e responsiva.

## 📂 Como Usar
1. Abra o arquivo `index.html` no seu navegador.
2. Faça login com a credencial configurada.
3. Escolha um nome para o seu projeto.
4. Cole o seu código HTML completo.
5. Clique em **Publicar** e receba seu link oficial.

## ⚠️ Configuração
Este projeto utiliza um **Personal Access Token (Classic)** do GitHub com permissões de `repo`. O token está configurado no script principal.
